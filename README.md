# Nextlum

__Controle Total. Consumo Inteligente.__

O presente projeto tem como objetivo o desenvolvimento do NEXTLUM, um 
sistema inteligente de automação residencial voltado para o controle e 
monitoramento de iluminação e temperatura de ambientes, com foco em 
eficiência energética, conforto e experiência do usuário. 

Diferentemente de sistemas convencionais de automação, o NEXTLUM não se 
limita ao simples acionamento remoto de dispositivos. A proposta é desenvolver 
uma solução inteligente capaz de otimizar o consumo de energia com base em 
variáveis ambientais e padrões de uso, agregando valor tecnológico e potencial de 
aplicação comercial.

## Objetivo Geral

Desenvolver um sistema de automação residencial inteligente capaz de controlar 
e monitorar iluminação e temperatura em tempo real, integrando hardware e 
aplicativo móvel. 


- Implementar controle manual e automático da iluminação; 
- Implementar controle térmico por meio de ventilador e/ou ar
condicionado; 
- Integrar sensores para coleta de dados ambientais; 
- Desenvolver aplicativo para controle e monitoramento remoto; 
- Implementar sistema de exibição de status em tempo real; 
- Desenvolver lógica de otimização para eficiência energética.
